# Node Project
## Installation

Step-by-step instructions on how to get the development environment set up.

1. Clone the repo

   ```sh
   git clone git@github.com:Shikhashukla283/sotbella.git
   cd sotbella

2.  Install NPM packages
    npm install

3.  Set up environment variables
    HOST='0.0.0.0'
    PORT= 3000
    USEREMAIL=''
    PASSWORD=''
    SECURITY_TOKEN=''
    CLIENT_ID=''
    CLIENT_SECRET=''

4. Start the Server
    npm run start
